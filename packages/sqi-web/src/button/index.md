# Button 按钮

## 基础使用

基本使用，type 预设 `primary` `default` `link` 三种基础按钮/颜色类型，可通过设置 `status` 来获得更多的颜色按钮。

当设置 `type='link;` 或 href 属性时，元素渲染则会变为 a 标签，且行为与 a 标签一致
